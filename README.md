### Задание 1

Master-Slave\
Запись данных происходит только на основном узле. Подчиненные узлы обычно используются для чтения данных.\
Если основной узел выходит из строя, система может стать недоступной для записи данных, пока не будет выполнено переключение на подчиненный узел, который станет новым основным узлом.\
\
Master-Master\
В master-master конфигурации записи могут происходить на любом ведущем узле, а чтение данных может выполняться с любого из узлов.\
В случае выхода из строя одного из ведущих узлов другие узлы могут продолжать обрабатывать как чтение, так и запись данных.
